# Инструкция для работы с Markdown

## Выделение текста

Для выделения текста полужирным, нужно занести его в (**) с двух сторон. Например:
**полужирный**
Для выделения текста курсивом, необходимо занести его в (*) с двух сторон. например:
*курсив*
